📊 Telco Customer Churn Analysis Dashboard

This project is part of my Data Science & Analytics Internship Program at Future Interns.

🔍 Project Objective

The goal of this project is to analyze customer churn behavior and identify factors affecting customer retention in a telecom company.

📌 Key Analysis
Customer churn rate
Customer tenure analysis
Monthly charges analysis
Contract type comparison
Payment method insights
Customer retention trends
🛠️ Tools Used
Microsoft Power BI
Data Visualization
Business Analytics
💡 Key Insights
Customers with month-to-month contracts had higher churn.
Long-term customers showed better retention.
Electronic check users had increased churn risk.
Internet service type impacted customer behavior.
🛩️Outcome

This project improved my understanding of customer retention analytics and dashboard storytelling using Power BI.
